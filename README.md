mugbot
====

Mugbot Engine

# Client

## Dependencies

* open_jtalk
* aplay

## Setup

```
$ cd /path/to/mugbot
$ scp client/raspberrypi/* pi@192.168.11.100:/home/pi
```

change 192.168.11.10 to your ip address of mugbot

## Run

```
$ python3 /home/pi/mugbot_client.py &
```
